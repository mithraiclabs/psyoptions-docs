PsyOptions est un protocole d'options vanilla construit sur la blockchain Solana. Les principes qui ont guidé l'architecture et le développement du protocole principal sont la flexibilité et la composabilité. Elle est totalement sans confiance. Le protocole principal ne fait aucune hypothèse quant à la manière dont les options doivent être négociées ou tarifiées. Les options sont représentées en tant que jetons SPL, ce qui signifie qu'elles peuvent être échangées sur n'importe quel DEX qui supporte les jetons SPL.

V1 soutiendra 100 % garantis, le règlement d’actifs, les appels couverts et les mises en vente sécurisées.

## Options de trading

Pour acheter ou vendre une option, sélectionnez `Marchés` dans la barre de navigation. Vous pouvez ajuster la date d'expiration et l'actif sous-jacent. L'équipe recommande d'utiliser les marchés d'options BTC/USDC du 28 mai 2021.

Après avoir sélectionné les paramètres, la page sera remplie de données de Sérum s'il y a un carnet de commandes pour le marché indivual. Notez que chaque ligne représente un marché d'options différent et donc un marché sérum différent.

![PsyOrderbook](/img/beta-testing/PsyOrderbook.png)

### Acheter pour ouvrir

L'achat d'une option (appel/put) ne nécessite aucune garantie. Il s'agit simplement d'une transaction sur le carnet de commandes de Sérum pour acheter une option sur le marché ouvert.

### Vendre pour ouvrir

Vendre une option est un peu plus complexe car une option peut devoir être modifiée. Si le portefeuille connecté n'a pas assez d'options à vendre, alors les options seront écrites avant de vendre.

Lorsque les options sont écrites, le portefeuille aura son actif sous-jacent déduit lorsqu'il sera envoyé au programme pour modifier les options. Le montant déduit est la taille `du contrat`, qui peut différer selon les paramètres sélectionnés.

### Types de commande

Le marché et les ordres à durée limitée sont pris en charge.

## Vérification des positions ouvertes

Cliquez sur l'option "Ouvrir les positions" en haut à gauche pour afficher toutes les positions actuelles sur le marché des options.

![Positions ouvertes](/img/beta-testing/PsyOpenPositions.png)

### Exercice

Si l'option n'a pas expiré, cliquer sur le bouton `Exercice` vous demandera une transaction pour exercer l'option.

Le portefeuille propriétaire doit avoir assez de l'actif de cotation (`taille du contrat * prix de levée`) pour pouvoir exercer avec succès.

## Initialisation des marchés

Pour créer des jetons SPL d'option fongibles pour le trading sur les marchés d'options, il faudra initialiser un marché en fournissant les valeurs suivantes :

- Ressource sous-jacente
- Actif de devis
- Taille du contrat
- Prix de grève
- Expiration

### Comment initialiser un marché :

1. Cliquez sur Initialiser

![tempsnip_1](/img/beta-testing/tempsnip_1.png)

2. Sélectionnez les paramètres suivants : Date d'expiration (selon l'expiration de l'option), Ressource sous-jacente (à échanger dans l'option), Actif de devis (à fournir dans la négociation), Taille du contrat (taille de l'option), Prix de base (prix d'option)

![format@@0 tempsnip_2](/img/beta-testing/tempsnip_2.png)

3. Sélectionnez "Multi Strikes" si vous souhaitez lister plusieurs contrats à intervalles de prix de levée désignés

![format@@0 tempsnip_3](/img/beta-testing/tempsnip_3.png)

4. Cliquez sur "Initialiser le marché"
