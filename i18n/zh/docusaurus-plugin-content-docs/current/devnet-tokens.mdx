## Claiming Devnet Tokens - Walkthe
**仅仅测试软件时所有已经有 0 个值**

1. 请确保你在你的钱包中获得一些SOL(更多特定的文档即将来临)

2. 导航到 _Faucets_ 页面 <img src={useBaseUrl("/img/wallet-setup/navigate-to-faucets-page.png")} alt="navigate-to-faucet-page" />

3. 连接您最喜欢的 Solana 钱包！ <img src={useBaseUrl("/img/wallet-setup/faucets-page.png")} alt="click-to-connect-wallet" />

4. 选择要空投的令牌 <img src={useBaseUrl("/img/wallet-setup/select-token.png")} alt="select-token-for-airdrop" />

5. 批准空投交易 <img src={useBaseUrl("/img/wallet-setup/approve-airdrop-tx.png")} alt="approve-airdrop-tx" />

6. GET PAID 🤑 🤑 🤑 (DISCLAIMER all TOKENS HAVE 0 VALUE THESTING PURPOSSES ONLY) <img src={useBaseUrl("/img/wallet-setup/get-paid.png")} alt="get-paid" />
