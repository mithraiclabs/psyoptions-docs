## Claiming Devnet Tokens - Walkthrough

_For PSY tokens, please go through the same method, and refer to the above mint and faucet address._

1. Go to [Sollet.io](http://sollet.io/)

2. On the top right of Sollet, click "Mainnet Beta" and switch to "http://devnet.solana.com".

<img
  src="../static/img/changesolletNetwork.png"
  alt="changesolletNetwork"
  style={{ zoom: "33%" }}
/>

3. Click "Request Airdrop" and wait for 1 SOL to be deposited into your wallet.

   <img
     src="../static/img/requestAirdrop.png"
     alt="requestAirdrop"
     style={{ zoom: "50%" }}
   />

4. Click on the "+" symbol to add a new token

   <img
     src="../static/img/addToken.png"
     alt="addToken"
     style={{ zoom: "50%" }}
   />

5. Select "Manual Input" and fill in the following information:

   1. Token Mint Address: HinfVnJuzMtJsyuLE2ArYCChDZB6FCxEu2p3CQeMcDiF
   2. Token Name: USDC
   3. Token Symbol: USDC

   <img
     src="../static/img/addTokenInput.png"
     alt="addTokenInput"
     style={{ zoom: "33%" }}
   />

6. Click add, refresh the wallet and wait for the address to load.

   ![associatedTokenMetadata](../static/img/associatedTokenMetadata.png)

7. Copy the Associated Token Metadata.

8. Go to https://spl-token-ui.netlify.app/#/token-faucets

9. In the top right, change to "devnet"

   <img
     src="../static/img/faucetChangeNetwork.png"
     alt="faucetChangeNetwork"
     style={{ zoom: "33%" }}
   />

10. Click "Token Airdrop"

11. Fill in the following:

    1. Admin: Leave empty.
    2. Token Destination Address: Paste the copied Associated Token Metadata.
    3. Faucet Address: `BmaVN3Wut1k2MtrcsxjTn919GdBJ2gzVLTtbiybnAPnR`
    4. Amount: `1000000000000`

<img
  src="../static/img/faucetForm.png"
  alt="faucetForm"
  style={{ zoom: "33%" }}
/>

12. Click "Airdrop Tokens" and a Sollet Wallet window will pop-up.

13. Switch to the Sollet Wallet window and click "Connect".

    <img
      src="../static/img/solletConnect.png"
      alt="solletConnect"
      style={{ zoom: "33%" }}
    />

    14. Approve the transaction.

        <img
          src="../static/img/approveTransaction.png"
          alt="approveTransaction"
          style={{ zoom: "33%" }}
        />

14. Check your Sollet if you have received the airdropped tokens.

    <img
      src="../static/img/solletBalance.png"
      alt="solletBalance"
      style={{ zoom: "33%" }}
    />
